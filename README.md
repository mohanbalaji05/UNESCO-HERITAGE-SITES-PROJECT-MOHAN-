# Heritage Treasures: An In-Depth Analysis of UNESCO World Heritage Sites Using Tableau

## 📖 Overview
This project analyzes the UNESCO World Heritage Sites (2019) dataset using Salesforce Tableau. It transforms raw data into interactive dashboards and stories to provide meaningful insights into heritage site distribution, categories, endangered sites, and inscription trends.

## 🎯 Objectives
- Analyze UNESCO World Heritage Sites data.
- Create interactive Tableau visualizations.
- Build responsive dashboards and stories.
- Publish the project using Tableau Public.
- Integrate the dashboard with Flask.

## 🛠️ Technologies Used
- Salesforce Tableau Public
- Python
- Flask
- Microsoft Excel
- GitHub

## 📊 Visualizations
- Treemap
- Bar Chart
- Pie Chart
- Line Chart
- Map
- Stacked Bar Chart
- Interactive Dashboard
- Tableau Story

## 📂 Dataset
- UNESCO World Heritage Sites (2019)
- Source: Kaggle

## ✨ Features
- Interactive Dashboard
- Story-based Data Analysis
- Country, Region, Category & Danger Filters
- Tableau Public Publishing
- Flask Integration
